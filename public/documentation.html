MOVIES REST API DOCUMENTATION

MOVIES ENDPOINTS

Business Logic Url HTTP Method Request body data Response body data
Get all movies to the user \movies GET None A JSON object data holding all movies
Get a movie by title to the user \movies\[TITLE] GET A request parameter of the movie title A JSON data object holding the movies for the request title
Get a genre by title or name of movies \movies\genre\[NAME/TITLE]\ GET A request parameter of the movie title or name A JSON data holding the genre for the request title
Get a movie director by title or name of movies \movies\director\[NAME/TITLE]\ GET A request parameter of the movie director name A JSON data object of director bio, birth year, death year
